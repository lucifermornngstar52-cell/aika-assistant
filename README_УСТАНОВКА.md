# 🌟 Айка — AI Ассистент | Инструкция по установке

## Что нужно для сборки

1. **Flutter SDK** (3.0+) — https://flutter.dev/docs/get-started/install
2. **Android Studio** или **VS Code** с Flutter расширением
3. **Android телефон** или эмулятор (Android 6.0+, API 23+)
4. **Java JDK 17** — https://adoptium.net

---

## 🚀 Быстрый старт

### Шаг 1 — Клонируй/распакуй проект
```bash
cd aika_assistant
```

### Шаг 2 — Установи зависимости
```bash
flutter pub get
```

### Шаг 3 — Подключи телефон
Включи **Режим разработчика** на телефоне:
- Настройки → О телефоне → нажми 7 раз на "Номер сборки"
- Настройки → Для разработчиков → Включи "Отладка по USB"

### Шаг 4 — Запусти
```bash
flutter run
```

Или собери APK:
```bash
flutter build apk --release
```
APK будет в: `build/app/outputs/flutter-apk/app-release.apk`

---

## 📱 Установка APK на телефон

1. Перенеси `app-release.apk` на телефон
2. Открой файловый менеджер → найди APK
3. Нажми → "Установить" 
4. Разреши установку из неизвестных источников если спросит

## 🎨 Добавить кастомный аватар

1. Возьми изображение Айки (PNG с прозрачным фоном)
2. Назови его `aika_avatar.png`
3. Положи в папку `assets/images/`
4. Пересобери приложение

## 🗣️ Голосовые команды

| Команда | Действие |
|---------|----------|
| "Открой YouTube" | Запускает YouTube |
| "Включи фонарик" | Включает фонарик |
| "Выключи фонарик" | Выключает фонарик |
| "Увеличь громкость" | Повышает звук |
| "Открой Telegram" | Запускает Telegram |
| "Открой TikTok" | Запускает TikTok |
| "Сколько заряда?" | Показывает % батареи |
| "Открой настройки" | Настройки телефона |
| "Найди [запрос]" | Поиск в Google |

## 🛠 Проблемы?

**Ошибка микрофона**: Проверь разрешения в Настройки → Приложения → Айка → Разрешения

**Ошибка AI**: Проверь интернет соединение. Можно переключить между Gemini и GPT-4o в Настройках

**Аватар не отображается**: Добавь `aika_avatar.png` в `assets/images/` — пока показывается встроенный анимированный аватар
